# Game2048
2048小游戏
## 项目介绍  
- 使用原生的Android控件实现的2048游戏
  - 游戏布局继承自GridLayout，通过添加事件监听来实现游戏
  - 每一个数字布局继承自FrameLayout，内嵌TextView实现
